Privacy Policy

Effective date: February 13, 2026

Overview

PPL Checkride App collects limited first-party usage data to improve learning outcomes, conversion from free to premium, and premium engagement. We do not sell personal data to ad networks.

Data We Collect

  • App interaction events (for example: mode started, scenario viewed, scenario rated, mock started/completed).
  • Technical identifiers used for analytics and session continuity (for example: anonymized browser ID and session ID).
  • License status context for premium features (hashed license identity only, never the raw license key).

Data We Do Not Collect in Analytics

  • Raw license keys.
  • Freeform answer text or model answer text.
  • Personal contact details from app analytics payloads.

Why We Use Data

  • Measure feature adoption and usage quality.
  • Understand difficult scenarios and improve content.
  • Monitor free-to-premium conversion and paid retention.
  • Troubleshoot sync and platform reliability issues.

Your Choices

You can disable usage analytics in the app via the Usage Analytics On/Off control. If disabled, new analytics events are not sent and pending event queue data is cleared locally.

Data Retention

We retain operational analytics and progress data only as long as needed for product operations, support, and performance analysis. Data deletion can be requested by support using available identity keys.

Contact

For privacy requests, contact: support@pplcheckride.com
